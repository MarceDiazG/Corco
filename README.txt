Setup automation environment for the JS-Mocha-Yadda framework:


Prerequisites:
on Mac: Developers tools of Mac (iTunes)
Access to TicketWeb environments (to run test cases)


Java 7:
http://www.oracle.com/technetwork/es/java/javase/downloads/jdk7-downloads-1880260.html


Python:
https://www.python.org/downloads/release/python-2712/


Node:
https://nodejs.org/download/release/v5.12.0/




go to nodejs folder and run:
npm install


to execute Test cases:
on Mac or Linux:
./node_modules/.bin/_mocha  test.js --timeout=0
on Windows:
mocha  test.js --timeout=0
